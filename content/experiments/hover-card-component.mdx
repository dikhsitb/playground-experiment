## Overview

Hover cards that appear instantly are annoying — you trigger them while moving the cursor across the page. This version adds a 300ms open delay and immediate close on mouse leave.

## Key Behavior

- **Open delay**: 300ms (configurable)
- **Close delay**: 0ms (immediate)
- **Position**: Auto-flip to stay in viewport
- **Keyboard**: Focusable, opens on focus, closes on blur

## Implementation Sketch

```tsx
const OPEN_DELAY = 300;

function useHoverDelay() {
  const [isOpen, setIsOpen] = useState(false);
  const timeoutRef = useRef<NodeJS.Timeout>();

  const onMouseEnter = () => {
    timeoutRef.current = setTimeout(() => setIsOpen(true), OPEN_DELAY);
  };

  const onMouseLeave = () => {
    clearTimeout(timeoutRef.current);
    setIsOpen(false);
  };

  return { isOpen, onMouseEnter, onMouseLeave };
}
```

- Card uses a subtle border instead of shadow — matches the minimal aesthetic
